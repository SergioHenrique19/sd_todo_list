## Especificação
  - Node v16 LTS

## Como executar
  - Para executar o client, acesse a pasta frontend, instale as dependências e execute o projeto
  ```bash
  cd frontend
  npm install
  npm run start
  ```

  - Para executar o server, acesse a pasta backend, instale as dependências e execute o projeto
  ```bash
  cd backend
  npm install
  npm run start
  ```
